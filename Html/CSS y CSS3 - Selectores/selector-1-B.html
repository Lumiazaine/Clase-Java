Selector universal con Box Sizing

Como segundo ejemplo vamos a utilizar el selector universal "*" con una nueva propiedad llamada "box-sizing".

El Box Sizing

La propiedad "box-sizing" nos permite incluir el padding y el border de un elemento en el total del "width" y del "height".

Por defecto el "width" y el "height" de un elemento se calcula así:

width + padding + border = "width" de un elemento

height + padding + border = "height" de un elemento
