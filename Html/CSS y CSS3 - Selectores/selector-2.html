Selector de Id

Un selector de tipo Id:

#miid

Selecciona un elemento único de nuestra página web .

Esta característica especial nos debe hacer preguntarnos a la hora de decidir su uso si es absolutamente necesario.

Los selectores de id son rígidos y no pueden ser reutilizados ya que son "propiedad" de un sólo elemento.

Lo utilizaremos para seleccionar y/o estilizar un solo elemento específicamente.
