Posicionamiento "Static"

Todos los elementos en HTML tienen por defecto su propiedad "position" establecida en "static":

position:static;

Esto quiere decir que si ponemos esta propiedad CSS en cualquier elemento (por ejemplo en un "div") no vamos a conseguir ningún cambio.

Cuando el navegador renderiza el HTML interpretando el código HTML de arriba hacia abajo y los elementos (de linea y de bloque) van ocupando sus posiciones naturales. Esto se conoce como "flujo normal".

Por ejemplo los párrafos anteriores que están dentro de etiquetas "p" se van renderizando en orden y como son elementos de bloque ocupan toda la línea.

Sin embargo si estuvieramos empleando etiquetas de línea (por ejemplo un "span", un "a", una imagen...) estos elementos irían posicionándose uno al lado de otro como en este ejemplo:

Por lo tanto si empleamos etiquetas de bloque y etiquetas de línea conjuntamente irán ocupando su posición natural en la pantalla.

Para resumir podemos decir que un elemento cuya propiedad "position" es "static" se posiciona en el body siguiendo un "flujo normal" y que ese elemento NO ESTÁ POSICIONADO.
