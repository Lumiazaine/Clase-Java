Posicionamiento "Relative"

Para posicionar un elemento con posicion "relative" utilizaremos la siguiente propiedad CSS:

position:relative

Un elemento que está posicionado relativamente modifica su posición "natural" de renderización con las propiedades: top, right, bottom y left.

El valor que le demos a estas propiedades situará al elemento en una posición relativa a la posición que hubiera ocupado este elemento si hubiera sido estático.
